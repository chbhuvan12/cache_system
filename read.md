# Cache System (LRU-Based)

A **high-performance in-memory cache system** implemented in Python, inspired by
real-world cache designs used in large-scale systems (Google, Redis, Memcached).

This project focuses on **O(1) get/put operations**, **LRU eviction**, and
clean system design.
